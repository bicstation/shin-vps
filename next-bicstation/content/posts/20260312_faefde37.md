## フォートナイトの協力モード「世界を救え」が無料化！今すぐ始められる！

人気ゲーム「フォートナイト」の協力型PvEモード「世界を救え」が、ついに基本プレイ無料化されます！これまで有料だったモードが無料で楽しめるようになるということで、フォートナイトをプレイしたことがない人や、バトルロイヤルは苦手だけど協力プレイが好きという人にとって、今が始める絶好のチャンスです。

### 「世界を救え」ってどんなゲーム？

「世界を救え」は、フォートナイトの世界観を舞台に、仲間と協力してモンスターの群れに立ち向かうモードです。拠点を建設したり、武器をクラフトしたり、フォートナイトならではの建築要素も楽しめます。バトルロイヤルとは違って、じっくりと戦略を練ってプレイしたい人におすすめです。

### 無料化で何が変わるの？

4月16日から「世界を救え」は誰でも無料でプレイできるようになります。事前登録をすると、ローンチ時に限定ヒーローなどのゲーム内報酬がもらえます。既存のプレイヤーには、これまでのプレイへの感謝として、スーパーチャージャーやV-Bucksなどの特典が配布される予定です。

### 事前登録の方法は？

フォートナイト公式サイトから事前登録できます。登録は簡単で、メールアドレスを入力するだけです。

[世界を救え- 4月16日(米国時間)から無料でプレイ](https://www.epicgames.com/store/ja/p/fortnite-save-the-world)

### フォートナイトを始めるには？

フォートナイトは、PC、Nintendo Switch、PS5、Xbox Series X|S、PS4、Xbox One、Androidなど、様々なプラットフォームでプレイできます。それぞれのプラットフォームでフォートナイトをダウンロードし、アカウントを作成すれば、すぐにプレイを始めることができます。

## フォートナイトを始めるメリット

フォートナイトは、無料で楽しめるだけでなく、様々な魅力があります。

*   **友達と協力プレイ**: 仲間と協力して、強敵に立ち向かう楽しさは格別です。
*   **建築要素**: フォートナイトならではの建築要素は、戦略性を高め、プレイの幅を広げます。
*   **豊富なスキン**: 個性的なスキンを集めて、自分だけのキャラクターを作り上げることができます。
*   **定期的なアップデート**: 定期的に新しいイベントやアイテムが追加され、飽きさせない工夫が凝らされています。

## フォートナイトをプレイするのに必要なPCスペック

フォートナイトを快適にプレイするためには、ある程度のPCスペックが必要です。

| 項目        | 推奨スペック                               |
| ----------- | ------------------------------------------ |
| CPU         | Intel Core i5-7300U 3.5 GHz / AMD Ryzen 3 3300U |
| GPU         | NVIDIA GeForce GTX 960 / AMD Radeon R7 370 |
| メモリ      | 8GB                                        |
| ストレージ   | 25GB以上の空き容量                           |

もし、PCのスペックが足りない場合は、グラフィック設定を下げたり、解像度を下げたりすることで、プレイできるようになる場合があります。

## まとめ

フォートナイトの協力モード「世界を救え」の基本プレイ無料化は、フォートナイトを始める絶好のチャンスです。友達と協力して、モンスターの群れに立ち向かう楽しさを、ぜひ体験してみてください。
