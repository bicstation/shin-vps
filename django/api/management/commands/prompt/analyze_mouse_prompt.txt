【マウスコンピューター専用・解析ルール】
1. CPUの判別:
   - 型番に「A」があれば AMD製CPU (例: A4-A5U)
   - 型番に「I」または「i」があれば Intel製CPU (例: I5U, I7G)
2. GPU(グラフィックス)の判別:
   - 「G-TUNE」「NEXTGEAR」はゲーミング機。型番内の「G60」「G50」等は RTX 4060 / RTX 3050 等の専用GPU搭載。
   - 「MousePro」「DAIV (R4)」「mouse (A4/B4)」等で、型番末尾が「U」なら「CPU内蔵グラフィックス」の可能性が高い。
3. 画面サイズ（ノートPCの場合）:
   - 「A4」「B4」「G4」は14インチ液晶。
   - 「A5」「B5」は15.6インチ液晶。
4. ブランド・筐体特徴:
   - 「G-TUNE」: 黒基調の本格ゲーミング。
   - 「NEXTGEAR」: 青いロゴ、ピラーレス、LED、若年層向けブランド。
   - 「DAIV」: クリエイター向け。色再現性が高い。
   - 「MousePro」: 法人・ビジネス向け。
