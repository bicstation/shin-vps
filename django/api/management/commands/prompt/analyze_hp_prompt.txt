【HP(ヒューレット・パッカード)専用・解析/推論ルール】

1. シリーズ（ブランド）別の性格判定:
   - 「OMEN」: ハイエンドゲーミング。冷却性能（Tempest Cooling）と高リフレッシュレートを強調。
   - 「Victus」: カジュアルゲーミング。コスパとシンプルなデザイン（白・青系）を強調。
   - 「Spectre」: プレミアム2-in-1。削り出しアルミ筐体、有機EL、デザインの美しさを強調。
   - 「Envy」: 上位スタンダード/クリエイター。画面の色再現性と質感を強調。
   - 「Pavilion / HP Laptop」: 実用スタンダード。信頼性と日常使いの快適さを強調。

2. CPU/GPUの判別補助:
   - 型番に「a」が含まれる (例: 16-ag..., 14-fa...): AMD製 CPU（Ryzen）と判定。
   - 型番に「w / f / d」等が含まれる (例: 16-wf..., 14-fc...): Intel製 CPU（Core Ultra / Core i）と判定。
   - 「Transcend」モデル: 薄型軽量ハイエンド。score_portable を通常ゲーミングより+15点加算せよ。

3. 画面サイズの特定:
   - シリーズ名の直後の数字（14, 15, 16）を液晶サイズとして確定。

4. AI PC判定:
   - 「Core Ultra」または「Ryzen 8000/AI 300」搭載機は is_ai_pc: true とし、NPUによる背景ぼかし等の効率化に言及せよ。
