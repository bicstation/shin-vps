44年のキャリアで、私は「エクセル作業が遅い」という理由だけでプロジェクトが停滞する場面を何度も見てきました。
エンジニアにとって、Officeソフトは「清書ツール」ではありません。データを素早く処理し、論理的な文書を構成するための「武器」です。

Vol.13では、マウスへの依存を断ち切り、指先の速度をそのままアウトプットの速度に変える技術を習得します。

## 1. マウスを捨てるための『Alt』キー戦術

Officeソフトの上部にある「リボン」メニューをマウスでクリックするのは、論理的なロスです。

- **アクセスキーの活用**: `Alt` キーを一度押してください。メニューにアルファベットが表示されます。
    - 例（Excel）: `Alt` → `H` → `A` → `C` （セルを中央揃えにする）
- **脊髄への記憶**: よく使う操作のキーシーケンスを身体で覚えます。これにより、メニューを「探す」という視認プロセスの時間をゼロにできます。

## 2. Excel：表計算を『データエンジニアリング』に変える

Excelは単なる表ではなく、簡易的なデータベースとして扱います。

- **Ctrl + 矢印キー**: データの末尾まで瞬時にジャンプする。
- **Ctrl + Shift + L**: フィルタのオン・オフ。マウスでリボンを探す必要はありません。
- **絶対参照と相対参照 ($)**: `F4` キーで一発切り替え。数式をコピーする際の論理ミスを防ぎます。
- **値貼り付け（Alt + E + S + V）**: 書式を汚さず、純粋なデータだけを扱うための「最頻出」コマンドです。

## 3. Word：『構造化』による文書作成

Wordが使いにくいと感じる最大の理由は、多くの人が「見た目」から作り始めるからです。

- **アウトラインモードの活用**: 最初に「章・節・項」の論理構造を固めます。
- **スタイルの適用（Ctrl + Alt + 1/2/3）**: 見出しをショートカットで定義します。これにより、目次生成や全体のレイアウト変更が1秒で完結します。
- **ナビゲーションペイン**: 左側に常に構造を表示させ、思考の迷子を防ぎます。



## 4. クイックアクセスツールバーのカスタマイズ

どうしてもショートカットが長い、あるいは覚えにくい操作は、自分専用の「特等席」に配置します。

- **0.5秒のアクセス**: ウィンドウの最上部に、自分が必要とするコマンド（例：セルの結合解除、PDFとして保存）を配置し、`Alt + 1` `Alt + 2` などで呼び出せるようにします。

## 結論：Office操作は『スポーツ』である

ショートカットを極めたあなたの指先は、思考の速度を追い越そうとするはずです。
「どのボタンだっけ？」と迷う暇を与えず、論理をそのままシートや文書に反映させる。この「道具との一体化」が完了したとき、事務作業はもはやあなたの時間を奪う敵ではなく、あなたの能力を証明するフィールドへと変わります。
