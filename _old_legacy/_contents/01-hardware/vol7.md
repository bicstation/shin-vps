国内BTOパソコンを語る上で、**「パソコン工房（iiyama PC）」**と**「ドスパラ（Galleria）」**を避けて通ることはできません。

両者は「最新パーツの採用速度」と「圧倒的なコストパフォーマンス」でしのぎを削っています。自作PCに近い自由度を持ちながら、メーカー製としての保証も受けられる。このバランスをどう読み解くかが、賢い調達の鍵となります。

## 1. パソコン工房：信頼の「iiyama」ブランドと地方ネットワーク

パソコン工房の最大の武器は、かつてモニターブランドとして名を馳せた「iiyama」の名を冠する**iiyama PC**の安定感です。

- **多種多様なラインナップ**: ビジネス向けの「SOLUTION∞」、クリエイター向けの「SENSE∞」、ゲーミングの「LEVEL∞」と、用途ごとにマザーボードのグレードまで細かく最適化されています。
- **実店舗の圧倒的密度**: 全国の主要都市に店舗を構えており、「物理的に持ち込んで即日修理・診断」ができる安心感は、地方在住のユーザーにとって最大の冗長化（バックアップ）となります。

## 2. ドスパラ：圧倒的な「物流速度」と独自筐体の進化

ドスパラの最大の特徴は、注文から出荷までの**「異常な速さ」**です。

- **最短当日出荷**: 「今すぐ演算リソースが必要だ」という状況において、ドスパラの物流システムは最強のソリューションです。
- **Galleriaの設計思想**: ゲーミングブランド「Galleria（ガレリア）」の現行筐体は、冷却効率とメンテナンス性を極限まで高めた独自設計。前面のゲート状の吸気口は、日本の多湿な環境でも効率的に外気を取り込めるよう計算されています。

## 3. ロジック比較：どちらを選ぶべきか？

両者の選択に迷った時、Mayaは以下の論理フィルタを推奨します。

| 項目 | パソコン工房 (iiyama) | ドスパラ (Galleria/Raytrek) |
| :--- | :--- | :--- |
| **強み** | 信頼のiiyama品質・店舗数 | 圧倒的な納期・パーツ鮮度 |
| **価格帯** | セール時の値引き率が高い | 標準価格から常に攻めている |
| **サポート** | 対面での物理サポートが強い | 24時間電話＋WEB連携がスムーズ |
| **おすすめ** | 長く安定して使いたい地方ユーザー | 1分1秒でも早く最新機が欲しい人 |

## 4. パーツ選定の裏側：マザーボードの「質」

BTOパソコンのコストを抑える際、最も削られやすいのが「マザーボード」です。

- **ドスパラ**：ASRockなどの大手メーカーと共同開発した専用基板を採用することが多く、必要十分な機能を効率よく配置しています。
- **パソコン工房**：標準的な市販マザーボード（ASUSやMSI）を採用する傾向があり、将来的なパーツ交換やBIOS更新の自由度がわずかに高い場合があります。

## 結論：コスパとは「時間」と「安心」の総和である

単にパーツ代を合計して比較するのは、エンジニアの視点ではありません。

「トラブル時にどれだけ早く復旧できるか（MTTRの短縮）」を重視するなら、近所に店舗がある**パソコン工房**。「最新のGPUを最速で手に入れたい」という欲望とスピードを重視するなら**ドスパラ**。

この「物理的な距離」と「物流の速度」こそが、スペック表には載らない本当のコストパフォーマンスの正体です。
