# Principle 01

Reality First

---

SHIN CORE LINX は Reality を起点とする。

```text
Reality
↓
Meaning
↓
Authority
↓
Presentation
```

Presentation は Reality を変更してはならない。

# Principle 02

Meaning Before Runtime

---

Runtime は Meaning を実現するために存在する。

```text
Meaning
↓
Runtime
```

であり、

```text
Runtime
↓
Meaning
```

ではない。

---

Runtime の存在理由は常に Meaning から説明できなければならない。
