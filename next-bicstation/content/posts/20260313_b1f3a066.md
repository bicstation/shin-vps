## 期待のホラーゲームスタジオ、Bokeh Game Studioが大型プロジェクト始動！

「SILENT HILL」や「SIREN」といった数々の名作ホラーゲームを手掛けた外山圭一郎氏が率いるBokeh Game Studioが、新たな大型プロジェクトを開始し、開発のコアメンバー募集を開始しました。これは、2024年11月に発売されたデビュー作「野狗子: Slitterhead」の成功を背景に、さらなるゲーム開発への意欲を示すものと言えるでしょう。

### Bokeh Game Studioとは？

Bokeh Game Studioは、外山圭一郎氏、佐藤一信氏、大倉純也氏といった、ゲーム業界を代表するクリエイターたちが集結したスタジオです。彼らは、これまでに数々の革新的なゲームを生み出し、ホラーゲームのジャンルに新たな可能性をもたらしてきました。

「野狗子: Slitterhead」は、九龍城砦をイメージした街を舞台としたホラーアクションアドベンチャーで、その独特な世界観と緊張感あふれるゲームプレイが多くのゲーマーから支持を集めました。

### なぜ今回のプロジェクト始動は重要なのか？

今回の大型プロジェクト始動は、Bokeh Game Studioが単なる一発屋ではなく、継続的に高品質なゲームを開発していく意思があることを示しています。ホラーゲームファンにとっては、今後の新作が非常に楽しみなニュースと言えるでしょう。

また、ゲーム業界全体で見ても、才能あるクリエイターたちが集結し、新たな挑戦を続けることは、業界の活性化につながります。

### 募集職種は？どんな人材が求められている？

募集されている職種は、キャラクターモデラー、アニメーターなどのアーティスト職を中心に、ゲームデザイナー、プログラマー、プロダクション、マーケティングなど、多岐にわたります。

特に、アーティスト職は、Bokeh Game Studioのゲームのクオリティを支える重要な役割を担います。ゲームデザイナーやプログラマーは、ゲームの企画・開発において、創造性と技術力を発揮することが求められます。

Bokeh Game Studioは、ゲームに対する情熱と、高いスキルを持った人材を求めているようです。

### 今後の展望：ホラーゲームの未来を切り開くか？

Bokeh Game Studioは、ホラーゲームのジャンルにおいて、新たなスタンダードを確立する可能性を秘めています。外山圭一郎氏の独創的なアイデアと、才能あるクリエイターたちの技術力が融合することで、これまでにない斬新なゲーム体験が生まれるかもしれません。

今後のBokeh Game Studioの動向に注目し、彼らがどのようなゲームを開発していくのか、期待しましょう。
